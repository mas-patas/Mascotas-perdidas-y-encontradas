# Logo de Más Patas

## Ubicación del archivo
Coloca el archivo del logo en esta carpeta con el nombre: `logo.png`

## Formatos soportados
- PNG (recomendado con fondo transparente)
- SVG (también soportado, cambia la extensión en Header.tsx si usas SVG)

## Especificaciones recomendadas
- Formato: PNG con fondo transparente
- Tamaño: Mínimo 512x512px para buena calidad en todos los dispositivos
- Nombre del archivo: `logo.png`

## Notas
- El logo se mostrará automáticamente en el navbar
- En dispositivos móviles solo se mostrará el logo (sin texto)
- En desktop se mostrará el logo junto al texto "MÁS PATAS"
